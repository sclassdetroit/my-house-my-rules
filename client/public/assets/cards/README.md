Place the uploaded card template images in this folder with these exact filenames:

- `blank-black-card.png` for prompt cards
- `blank-white-card.png` for answer cards
- `black-logo-card.png` for hidden/card-back cards
- `adult-warning-card.png` for the optional After Dark warning image

The app has CSS fallback cards so it runs immediately if these images are missing. Replace the placeholders by copying your uploaded designs here with the filenames above.
