Place the uploaded card template images in this folder with these exact filenames:

- `blank-black-card.jpg` for prompt cards
- `blank-white-card.jpg` for answer cards
- `black-logo-card.jpg` for hidden/card-back cards
- `adult-warning-card.png` for the optional After Dark warning image

The original uploads are JPEG files, so the app uses `.jpg` paths for reliable hosting on Netlify.
