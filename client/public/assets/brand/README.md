Put your brand images in this folder with these exact filenames:

- `company-logo.png` shows inside the game interface.
- `app-icon.png` is used for the browser tab icon, web app manifest, and iPhone home-screen icon.

Recommended image sizes:

- `company-logo.png`: transparent PNG, about 800x300 or similar wide logo.
- `app-icon.png`: square PNG, 1024x1024 preferred.

After replacing these files, run the frontend build again and upload the new `client/dist` folder to Netlify.
